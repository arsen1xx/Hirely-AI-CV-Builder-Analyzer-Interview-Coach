# Hirely — AI-Powered CV Builder, Analyzer & Interview Coach 🚀

**Hirely** — це інноваційний Telegram-бот, розроблений для автоматизації процесу створення резюме та підготовки до технічних співбесід. Бот використовує потужності штучного інтелекту (Gemini API) для надання персоналізованих порад та генерації професійних документів.

## Основні можливості 🛠️

- **📊 AI CV Analyzer:** Завантажуйте своє резюме у форматі PDF, і бот проаналізує його за стандартами ATS (Applicant Tracking System), визначить ваш ATS Score, знайде помилки та дасть поради щодо покращення.
- **📄 PDF Generator:** Створюйте стильне та лаконічне резюме безпосередньо в чаті.
- **🎙️ Mock Interview:** Пройдіть симуляцію реальної співбесіди на будь-яку посаду (UI/UX Designer, Frontend, Backend тощо). Після 4 складних питань ви отримаєте розгорнутий фідбек від Senior HR зі своєю оцінкою.
- **👤 Профіль користувача:** Зберігайте дані про свій досвід, освіту та проєкти для швидкого редагування CV.

## Технологічний стек 💻

- **Мова:** Python
- **Бібліотеки:** `python-telegram-bot`, `fpdf2` (генерація PDF), `PyPDF2` (парсинг тексту).
- **AI Engine:** Groq API (модель Llama-3.3-70b-versatile).
- **Environment:** `python-dotenv` для безпечного керування токенами.
